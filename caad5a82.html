一.前期软件准备

vs code安装

Google搜索vs code

选择第一个直接安装，我这里选择Windows系统

jdk安装

Google搜jdk oracle

第二部分 实现功能

新建java文件，例如 na.java

public class na {
    public static void main(String[] args) {
        System.out.println("hello java");
    }
}

终端输入下面代码

javac na.java

java hello

即可得到输出

hello java

第三部分 数据类型

注解 (给人看的文字)

  • 单行注解示范：

    // 注解内容

  • 多行注解示范:
    /*
    第一行注解
    第二行注解
    */

Data Types

  • 整型: byte,short,int,long
  • 浮点数: float,double
  • 布林值：boolen
  • 字元： char
  • 字串： String

使用方法

在vscode里面创建DataTypeVarible.java文件，输入以下代码：

public class DataTypeVariable{
    public static void main(String[] args){
        // 在终端中打印各种类型的资料
        // 整型 预设为int 
        System.out.println(3);
        System.out.println(120);
        // 变量x的使用
        int x;
        x=3;
        System.out.println(x);
        x=120;
        System.out.println(x);
        // 长整型 数字的后面加上l
        System.out.println(2000000000L);
        // 什么形式，应该对应什么形式，所以应该为long y而不是int y
        long y=200000000000L;
        System.out.println(y);
        // 浮点数，预设为double类型 double
        System.out.println(3.14564965);
        // 单精度浮点数 数字后面加上 F 只能到小数点后7位数 float
        System.out.println(3.141595233F);
        // 布林值 boolean
        System.out.println(false);
        // 字元 char
        System.out.println('a');
        // 字符串 String
        System.out.println("hello 你好呀");
    }
}

终端执行javac DaTaTypeVarible,java进行编译

终端执行java DataTypeVarible.java运行代码
