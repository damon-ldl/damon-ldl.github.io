BeautifulSoup美化输出内容

利用BeautifulSoup对输出格式进行美化，BeautifulSoup是一个Python库，用于解析HTML或XML文件，以便于从中提取数据。它是一个强大的解析器，能够省去编写正则表达式的繁琐工作。BeautifulSoup支持多种解析器，包括Python标准库中的HTML解析器以及第三方解析器如lxml。

使用BeautifulSoup，你可以方便地定位到HTML或XML文件中的特定标签，并从这些标签中提取出你需要的信息。此外，它提供了一些实用的功能，如查找和修改文档内容。

要使用BeautifulSoup，你首先需要导入这个模块：from bs4 import beautifulsoup。然后，你可以创建一个BeautifulSoup对象，指定你要解析的内容和使用的解析器：soup = beautifulsoup(parse_content, 'parser')。其中，parse_content是你要解析的内容，'parser'是你选择的解析器。

代码解释：

href = x.a.get('href') 这行代码的意思是从HTML元素 x.a 中获取名为 ‘href’ 的属性值，并将其赋值给变量 href。

在这段代码中，x 是一个HTML元素对象，通常表示一个标签（如 <a>）。x.a 表示该元素的子元素 <a>。get('href') 是一个方法，用于获取指定属性的值。在这里，它获取的是 <a> 标签的 ‘href’ 属性值，这个值通常是一个URL链接。

抓取网页电影名称和电影地址

实现代码:

def get_url(url):
    print("抓取网址：", url)  # 打印正在抓取的网址
    headers = Agent_info()  # 获取请求头信息
    request = requests.get(url, headers=headers)  # 发送GET请求，携带请求头信息
    soup = BeautifulSoup(request.text, 'lxml')  # 使用BeautifulSoup解析网页内容
    pic = soup.find_all(attrs={'class': 'pic'})  # 查找所有具有'class': 'pic'属性的元素
    film_urls = []  # 电影详情地址列表
    for x in pic:
        href = x.a.get('href')  # 获取每个元素的a标签中的href属性值
        film_urls.append(href)  # 将href属性值添加到电影详情地址列表中
    movie_list = []  # 外国电影名字列表
    div_list = soup.find_all(attrs={'class': 'hd'})  # 查找所有具有'class': 'hd'属性的元素
    for each in div_list:
        movie = each.a.contents[3].text.strip()  # 获取每个元素的a标签中的第三个子元素的文本内容，并去除首尾空格
        movie = movie[2:]  # 从第三个字符开始取值
        movie_list.append(movie)  # 将取到的值添加到外国电影名字列表中

    return film_urls, movie_list  # 返回电影详情地址列表和外国电影名字列表
