faschat基本知识

命令行推理

使用FastChat库来加载并服务大型语言模型的指令

安装`pip3 install "fschat[model_worker,webui]"`
加载大模型推理 python -m fastchat.serve.cli --model-path /你的下载路径/chatglm2-6b
linux:python -m fastchat.serve.cli --model-path /root/LangChain-ChatGLM-Webui/model/chatglm2-6b
windows: python -m fastchat.serve.cli --model-path D:\project\LangChain-ChatGLM-Webui\model\chatglm2-6b

web实现

利用 FastChat 实现模型调用确实可以分为三个主要部分,OpenAI API 服务器启动步骤：

1.启动控制器（Controller）:

  • 使用 python3 -m fastchat.serve.controller 命令启动 FastChat 控制器。
  • 控制器负责管理请求的调度和分发，将用户的请求路由到适当的工作器（worker）。

2.启动模型工作器（Model Worker）:

  • 使用 python3 -m fastchat.serve.model_worker 命令启动模型工作器，并指定模型名称和路径。
  • 工作器是实际处理请求的地方，它们与后端模型服务进行交互并返回响应。

3.启动 RESTful API 服务器:

  • 使用 python3 -m fastchat.serve.openai_api_server 命令启动 RESTful API 服务器。
  • API 服务器提供了一个 HTTP 接口，允许外部客户端（如前端应用或其他服务）通过 RESTful API 发送请求。

每个部分都承担着不同的职责，共同构成了 FastChat 的架构：

  • 控制器：作为请求的入口点，负责协调和管理工作流程。
  • 工作器：执行实际的模型调用和处理逻辑。
  • API 服务器：提供一个标准的 HTTP 接口，使得其他系统能够与 FastChat 交互。

启动 controller

python -m fastchat.serve.controller --host 0.0.0.0

启动模型

# 必须是本地ip 
linux:python -m fastchat.serve.model_worker --model-path /root/LangChain-ChatGLM-Webui/model/chatglm2-6b --host 0.0.0.0 
windows:python -m fastchat.serve.model_worker --model-names chatglm2-6b --model-path D:\project\LangChain-ChatGLM-Webui\model\chatglm2-6b --host 0.0.0.0

openapi服务

python -m fastchat.serve.openai_api_server --host 0.0.0.0

gradio服务

# gradio服务对gradio版本有要求
linux:python -m fastchat.serve.gradio_web_server --host 0.0.0.0
windows:python -m fastchat.serve.gradio_web_server

服务默认端口是 7860，可以通过--port参数来修改端口，还可以通过添加--share参数来开启 Gradio 的共享模式，这样就可以通过外网访问 WebUI 服务

OpenAI API 接口实现

代码示例1：

前面三部分代码同时运行后，本地服务正在运行并且能够接收和处理OpenAI API格式的请求

#要求openai版本为0.28 text
import openai
# to get proper authentication, make sure to use a valid key that's listed in
# the --api-keys flag. if no flag value is provided, the `api_key` will be ignored.
openai.api_key = "EMPTY"
openai.api_base = "http://localhost:8000/v1"

model = "D:\\project\\LangChain-ChatGLM-Webui\\model\\chatglm2-6b"
prompt = "Once upon a time"

# create a completion
completion = openai.Completion.create(model=model, prompt=prompt, max_tokens=64)
# print the completion
print(prompt + completion.choices[0].text)

# create a chat completion
completion = openai.ChatCompletion.create(
  model=model,
  messages=[{"role": "user", "content": "Hello! What is your name?"}]
)
# print the completion
print(completion.choices[0].message.content)

代码示例2：

from langchain.chat_models import ChatOpenAI
from langchain.schema import HumanMessage

llm = ChatOpenAI(
    streaming=True,
    verbose=True,
    # callbacks=[callback],
    openai_api_key="none",
    openai_api_base="http://127.0.0.1:8000/v1",
    model_name="Qwen-7B-Chat"
)
instructions = """
你将得到一个带有水果名称的句子,提取这些水果名称并为其分配一个表情符号
在 python 字典中返回水果名称和表情符号
"""

fruit_names = """
苹果,梨,这是奇异果
"""

# 制作结合说明和水果名称的提示
prompt = (instructions + fruit_names)

# Call the LLM
output = llm([(content=prompt)])

print (output.content)
print("========")
print (type(output.content))
# 苹果: 🍎梨: 🥐奇异果: 🍓
# ========
# <class 'str'>

代码示例3

from langchain import PromptTemplate
from langchain.chat_models import ChatOpenAI
from langchain.schema import HumanMessage

llm = ChatOpenAI(
    streaming=True,
    verbose=True,
    # callbacks=[callback],
    openai_api_key="none",
    openai_api_base="http://localhost:8000/v1",
    model_name="chatglm2-2b"
)


# 提示词
template = """
我很想去{location}旅行，我应该在哪里做什么？
"""
prompt = PromptTemplate(
    input_variables=["location"],
    template=template,

)
# 说白了就是在提示词的基础上，把输入的话进行格式化方法输入，前后添加了一些固定词
final_prompt = prompt.format(location='安徽合肥')

print(f"最终提升次：{final_prompt}")
output = llm([HumanMessage(content=final_prompt)])
print(f"LLM输出结果：{output}")
